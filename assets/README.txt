MY DENTAL SAVINGS – Static Landing Site
=====================================
Files:
- index.html          (landing page)
- terms.html          (terms & conditions)
- privacy.html        (privacy policy)
- assets/logo.png     (provided logo)

Tailwind is loaded via CDN in each HTML file.

Brand Colors (from logo):
- Blue: #0796D1
- Teal: #06D1A1
- Sand (card): #E9DCC7

Optional Background Themes (use on <body> or sections):
1) Soft Dental Blue:   class="bg-[#F1F9FE]"
2) Mint Wash:          class="bg-[#E9FCF6]"
3) Sand Beige:         class="bg-[#F5EFE4]"  (pairs with card look)
4) Gradient Blue→Teal: class="bg-gradient-to-b from-[#F1F9FE] to-[#E9FCF6]"
5) Slate Contrast:     class="bg-slate-50"

Change hero texture: the 'icon-pattern' class adds a subtle tiled icon background.
